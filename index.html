FullDriveWorld

Auf dieser Seite kann FullDriveWorld als Web Start Anwendung heruntergeladen werden.

Anleitung

1. OpenWebStart herunterladen und installieren

2. JNLP-Datei hier herunterladen

3. Die JNLP-Datei per Doppelklick ausführen. Es werden Warnhinweise angezeigt; diese können getrost ignoriert werden, denn die App ist sicher.


Achtung: App funktioniert nur unter Windows
